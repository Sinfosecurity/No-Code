AI Prompt Cheat Sheet

30+ Ready-to-Use Prompts for Building Apps & Websites

The Golden Formula

WHAT + WHO + FEATURES + DESIGN

Always include these 4 elements for the best results

Pro Tips for Better Prompts

  • Be specific: "blue and white colors" is better than "nice colors"
  • List features: Use numbered lists so nothing gets missed
  • Mention "single HTML file": This keeps things simple for copy-paste
  • Say "mobile-friendly": So it works on phones too
  • Ask for "modern design": Gets you rounded corners, shadows, and clean layouts
  • Include sample data: Give real examples so the app looks complete

Customization Prompts (Use After Building)

DESIGN

Change Colors

Change the color scheme to [NEW COLORS]. Keep everything else the same. Make sure it still looks professional.
DESIGN

Add Dark Mode

Add a dark mode toggle button in the top-right corner. When clicked, switch between light and dark themes. Remember the user's preference.
FEATURE

Add WhatsApp Button

Add a floating WhatsApp button in the bottom-right corner that opens WhatsApp chat with the number +234 XXX XXX XXXX with a pre-filled message "Hi! I found you on your website."
FEATURE

Add Search

Add a search bar at the top that filters and highlights matching items as the user types. Show "No results found" if nothing matches.
FEATURE

Add Animations

Add smooth entrance animations to all sections. Elements should fade in and slide up as the user scrolls down the page. Also add hover effects on all clickable elements.
FEATURE

Make Mobile-Friendly

Make sure this app/website is fully responsive and looks great on mobile phones. Add a hamburger menu for navigation on small screens. Test all features at 375px width.

Fixing & Troubleshooting Prompts

FIX

Code Not Working

The code you gave me isn't working when I paste it into Replit. Can you check for errors and give me the complete, fixed code? Make sure it's a single HTML file that works on its own.
FIX

Layout Looks Broken

The layout looks broken - things are overlapping and not aligned properly. Can you fix the CSS to make sure everything is properly spaced, aligned, and looks clean on both desktop and mobile?
FIX

Feature Not Working

The [FEATURE NAME] feature isn't working correctly. When I [DESCRIBE WHAT YOU DO], [DESCRIBE WHAT HAPPENS INSTEAD]. Can you fix this and give me the complete updated code?
FIX

Start Over Clean

Let's start fresh. Please rebuild the entire app/website from scratch with all the features we discussed. Give me a complete, working single HTML file with no errors.
